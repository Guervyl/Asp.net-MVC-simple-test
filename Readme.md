Prueba C#
---------

## Esta prueba es una aplicación web ASP.net MVC. inlcuye:

- Entity framework.
- Validación de formularios

## ¿Que hace?
Insertar datos en una tabla proveedor y una tabla venta.
Y mostrar un reporte de las ventas por proveedor y permitir el filtro con fecha.

## Como ejecutar el codigo.
Para ejecutar el codigo hay que instalar los paquetes NuGet para poder instalar entity framework.

Y hay que modificar la cadena de la base de datos en los archivos de configuración de los proyectos con sus datos.

## DDL
En la carpeta *DDL* de la solución, se encuentra el codigo SQL para Sql server para crear las tablas necesarias.
